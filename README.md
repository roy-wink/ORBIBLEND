# ORBIBLEND

<p align="center">
  <img src="docs/img/orbiblend_logo.png" alt="ORBIBLEND logo" width="320">
</p>
 
**ORBIBLEND** is a Blender add-on for visualizing molecules and molecular orbitals from `.abo` files. It is designed especially for educational use: students can import a file, choose a molecular orbital, adjust the view, and render an image or animation from one focused Blender sidebar.

The add-on aims to make orbital visualization approachable for users who are new to Blender, while still leaving the generated Blender scene editable for users who want more control.

## Requirements

ORBIBLEND is developed for Blender 4.x and currently targets the Blender/Python versions listed in the add-on metadata. See the documentation for the recommended setup and installation steps.

## Repository overview

- `abo-addon/` — the Blender add-on package.
- `abo-files/` — sample `.abo` files for trying the add-on.
- `docs/` — Sphinx documentation source and assets.

## Project status

ORBIBLEND is usable and under active development. Feedback from students, educators, and Blender users is welcome, especially when something in the workflow is unclear for first-time users.
