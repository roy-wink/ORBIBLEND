Educational use
===============

ORBIBLEND is intended to support conceptual learning.  The add-on can make
attractive images, but its main value is helping students connect a mathematical
orbital surface to molecular shape, symmetry, and bonding.

Suggested learning goals
------------------------

Students can use ORBIBLEND images to practice identifying:

* positive and negative orbital lobes;
* nodal planes and nodal surfaces;
* bonding and antibonding overlap;
* orbital symmetry with respect to the molecular geometry;
* how orbital shape changes across a set of molecular orbitals.

Guidance for first-time Blender users
-------------------------------------

Students do not need to learn Blender modeling to use ORBIBLEND.  Recommended
rules for a beginner activity are:

#. Use only the ``ABO`` sidebar.
#. Press buttons from top to bottom.
#. Start with a sample ``.abo`` file.
#. Render still images before trying animations.
#. Ask for chemical interpretation first, visual polish second.

Making figures easier to interpret
----------------------------------

* Keep atoms and bonds visible for introductory exercises.
* Use transparency when the molecule is hidden inside an orbital surface.
* Rotate in small increments and stop when the important symmetry or node is
  clear.
* Use the same viewpoint when comparing orbitals.
* Avoid relying on color alone; mention lobe sign in captions or labels.

Classroom activity ideas
------------------------

Orbital matching
   Give students several rendered orbital images and ask them to match each one
   to a frame description or molecular-orbital label.

Bonding versus antibonding
   Render two orbitals from the same molecule and ask students to identify where
   constructive and destructive overlap appear.

Symmetry discussion
   Rotate an orbital until a nodal plane is visible, then ask students to predict
   how the orbital changes under reflection or rotation.

Accessibility notes
-------------------

For projected slides and printed handouts, use high-contrast colors and avoid
small differences in transparency as the only cue.  Captions should describe what
students are expected to notice, such as "node between the atoms" or "larger lobe
on oxygen".
